# OBS Forum Resource Submission

## Resource Title

Prompter Studio

## Tagline

Third-party teleprompter plugin for OBS Studio on Windows.

## Description

Prompter Studio adds an in-scene teleprompter source, a private operator preview, and a dedicated control dock. It supports continuous scrolling, manual paragraph presentation, OBS hotkeys, optional Xbox trigger control through XInput, text files, configurable typography, mirrors, and persistent settings.

## Requirements

- Windows 10 or Windows 11
- OBS Studio 64-bit

## Installation

1. Close OBS Studio.
2. Run `Prompter-Studio-Setup.exe`, or extract `Prompter-Studio.zip` into the OBS Studio installation directory.
3. Restart OBS Studio.
4. Add `Prompter Source` from the Sources menu and enable the Prompter Studio docks from View > Docks.
